LICENSE
-------

The TOPPERS/SafeG logo can be used for the following goals:

1.- In a product o service to show its relation with TOPPERS/SafeG.

2.- Inside a machine or device to show that TOPPERS/SafeG is used.

3.- In a document or presentation slides to introduce TOPPERS/SafeG.

For any other usage, please contact with the TOPPERS secretary office
(http://www.toppers.jp/contacts.html).

DATA
----

The TOPPERS/SafeG logo was created using Inkscape (http://inkscape.org/).
The file "safeg-logo.svg" is the main source file. The fonts used within
the SafeG logo are "courier" for "SafeG"; and "arial" for "real-time
embedded dual-os system".
